2026 年 5 个最佳 Agent Skills 平台推荐

如果你正在使用 Claude Code、Cursor 或其他 AI 编程助手，却还在手动编写每个提示词，你正在错过 Agent Skills 带来的生产力革命。这篇文章为你精选 5 个最值得关注的 Agent Skills 平台，让你的 AI Agent 从”对话助手”升级为”专业工作流执行者”。

1. NanoSkill.ai — 营销人的技能武器库

官网： https://nanoskill.ai

NanoSkill.ai 是专注营销领域的 Agent Skills 聚合平台，如果你是增长黑客、数字营销经理或技术型创业者，这是你的首选平台。

核心数据

指标 数据
技能总数 2,400+
周安装量 50,000+
支持平台 10 个（Claude Code、Cursor、Windsurf 等）
活跃技能率 98%

主要功能

NanoSkill 不是泛泛的 AI 工具目录，而是围绕真实营销工作流精选技能：

  • SEO 优化：关键词研究、内容优化、技术 SEO 审计

  • 广告投放：Google Ads、Meta Ads、LinkedIn Ads 活动规划与优化

  • 内容营销：博客写作（针对 Google 排名和 AI 引用平台双重优化）、社交媒体文案

  • 潜在客户开发：B2B 线索挖掘、人物搜索、陌生开发信自动化

  • 数据分析：YouTube 广告分析、营销报表生成

热门技能案例

创意内容生成技能（163K stars）：生成 ASCII 艺术、手绘风格图表和视觉设计，提升项目视觉吸引力。

广告活动规划师（156 stars）：通过智能规划、出价策略、预算分配和创意优化，帮助管理 Google Ads、Meta Ads 和 LinkedIn Ads 活动。

AI 博客写作技能（818 stars）：大规模生成、优化和审核部落格内容，产出针对 Google 排名和 AI 引用平台双重优化的文章。

适用人群

  • 技术型营销人员：需要自动化 MarTech 工作流的增长团队

  • 创业公司创始人：预算有限，需要用 AI 替代专职营销团队

  • 营销机构：服务多个客户，需要标准化交付流程

  • 内容创作者：管理多个分发渠道，需要内容再利用自动化

独特优势

  • 质量把控：精选与测评机制，98% 活跃技能率远超行业平均

  • 营销专注：按实际营销任务（而非技术分类）组织内容

  • 跨平台聚合：整合 2,400+ 来自不同创作者的技能，提供统一发现平台

  • 繁体中文支持：提供 /tw 版本，对华语市场友好

2. ClawHub.ai — OpenClaw 的官方技能商店

官网： https://clawhub.ai

ClawHub 是 OpenClaw（220K+ GitHub stars）的官方技能注册中心，被称为”AI Agents 的 npm”。如果你正在使用 OpenClaw 这个开源 AI Agent 框架，ClawHub 是你的必备平台。

核心数据

指标 数据
技能总数 3,286+
总下载量 数百万次
技术架构 向量搜索 + 语义检索
安全机制 VirusTotal 扫描

主要功能

ClawHub 提供完整的技能生命周期管理：

  • 发布与版本控制：开发者可以发布技能包，使用语义版本号（semver）管理版本

  • 向量搜索：基于语义的技能发现，而非简单的关键词匹配

  • CLI 工具：通过命令行快速安装技能（clawhub install <skill-name>）

  • 社区评分：星标数、下载量和社区评论帮助评估技能质量

  • 安全扫描：集成 VirusTotal 对技能进行安全检查

热门技能案例

Capability Evolver（35,581 下载）：让 AI Agent 从任务执行者转变为主动合作伙伴，持续改进自身能力。

self-improving-agent（15,962 下载，132 stars）：使 Agent 能够从执行历史中学习，优化未来表现。

GitHub 集成技能：监控 PR 通知、自动代码审查、Issue 管理，实现完整的 GitHub 工作流自动化。

适用人群

  • OpenClaw 用户：使用 OpenClaw 框架的开发者和团队

  • 技能开发者：希望发布和分发自己技能包的创作者

  • 企业团队：需要私有技能库和版本管理的组织

  • 自动化爱好者：追求智能家居、消息处理等跨平台自动化的用户

独特优势

  • 官方生态：OpenClaw 团队构建和维护，与框架深度集成

  • 技术先进：向量搜索技术提供更智能的技能发现体验

  • 开发者友好：完整的 CLI 工具链，类似 npm 的使用体验

  • 社区驱动：3,000+ 社区贡献的技能，覆盖从编程到智能家居的广泛场景

Skills.sh 是 Vercel 于 2026 年 1 月 20 日推出的官方 Agent Skills 目录和排行榜平台，专注于为 AI 编程助手提供可发现、可共享的程序化知识。

核心数据

指标 数据
技能总数 200+ 精选技能
累计安装 573,986+
支持平台 Cursor、Claude Code、GitHub Copilot、Codex、Windsurf 等
安装方式 一键命令安装

主要功能

Skills.sh 提供简洁高效的技能发现和安装体验：

  • 排行榜系统：实时展示热门技能、趋势技能和全时段排行

  • 一键安装：通过 npx skills add <owner/repo> 命令快速安装

  • 使用统计：透明展示每个技能的安装量和使用趋势

  • GitHub 集成：直接从 GitHub 仓库安装技能，支持开源生态

  • 跨平台兼容：与主流 AI 编程助手无缝集成

热门技能案例

Vercel React Best Practices（131,000+ 安装）：Vercel 工程团队的 React 和 Next.js 性能优化模式，用于构建高性能 Web 应用。

Trail of Bits Security Skills：来自知名安全公司的安全审计和分析技能，包含 CodeQL、Semgrep 等静态分析工具。

Superpowers：为 AI Agent 带来工程纪律，确保生成的代码不仅”能工作”，而且”做对了”。

适用人群

  • 前端开发者：需要 React、Next.js、Tailwind 最佳实践的团队

  • 全栈工程师：使用 AI 编程助手加速开发的个人开发者

  • 安全工程师：需要安全审计和代码分析工作流的专业人士

  • 技术团队：希望标准化 AI Agent 工作流的组织

独特优势

  • 官方背书：Vercel 官方维护，质量有保障

  • 精选策略：200+ 精选技能，而非海量低质内容

  • 实时数据：透明的使用统计和趋势分析

  • 开源友好：直接从 GitHub 安装，支持社区贡献

  • 零配置：一行命令完成安装，无需复杂设置

官网： https://agenticskills.io

AgenticSkills.io 是一个综合性平台，不仅提供 Agent Skills 目录，还整合了 200+ MCP（Model Context Protocol）服务器，为 AI Agent 提供外部工具和 API 集成能力。

核心数据

指标 数据
Agent Skills 数百个精选技能
MCP 服务器 200+ 官方和社区服务器
支持平台 Claude Code、Codex、Cursor、Gemini CLI 等 18+ 平台
内容类型 技能 + MCP 服务器 + 规范文档

主要功能

AgenticSkills.io 提供双轨道的能力扩展：

  • 技能目录：涵盖营销、开发、设计、安全等多个领域的 Agent Skills

  • MCP 服务器目录：GitHub、Stripe、AWS 等官方 MCP 集成，连接 AI Agent 到外部系统

  • 对比功能：并排比较不同技能和服务器的特性

  • 规范文档：完整的 Agent Skills 格式规范和最佳实践

  • 社区讨论：GitHub 和 Discord 社区支持

特色技能案例

Marketing Skills Bundle by Corey Haines：综合营销技能包，涵盖 CRO、文案、SEO、分析、付费广告、生命周期邮件和增长工程。

shadcn/ui Skill：教 Agent 使用 shadcn/ui CLI v4、注册系统、组件模式和 Tailwind v4 配置。

Next.js Performance Optimization：Next.js 性能优化和最佳实践，涵盖服务器组件、路由和数据获取。

适用人群

  • 全栈开发者：需要同时使用技能和 MCP 服务器的开发者

  • 企业团队：需要连接 AI Agent 到内部系统（CRM、分析平台等）

  • 技能创作者：希望学习规范并发布高质量技能的开发者

  • 技术决策者：需要评估和对比不同 Agent 能力扩展方案的管理者

独特优势

  • 双料目录：同时提供技能和 MCP 服务器，满足不同扩展需求

  • 官方集成：收录 GitHub、Stripe、AWS 等官方 MCP 服务器

  • 规范权威：基于 Anthropic 发布的开放标准，提供完整规范文档

  • 对比工具：帮助用户快速评估和选择合适的技能

  • 生态完整：从技能发现、安装到社区讨论的完整闭环

两个关键建议

1. 从项目上下文开始

在安装任何技能之前，先创建 .agents/product-marketing.md 文件，描述你的产品、目标受众和定位。大多数高质量技能会读取这个上下文文件，从而提供更精准、更符合你品牌调性的输出。这个简单的步骤可以让技能输出质量提升 3-5 倍。

2. 少即是多：每次任务 2-3 个技能

根据 SkillsBench 研究，同时加载过多技能会稀释 Agent 的注意力。建议每次任务只安装 2-3 个最相关的技能，完成后再切换到其他技能。这样可以保持 Agent 的专注度和输出质量。

不同人群的下一步行动

如果你是营销人员或创业者，从 NanoSkill.ai 开始，安装 SEO、广告或内容营销相关技能，立即提升营销自动化能力。

如果你是前端或全栈开发者，访问 Skills.sh，安装 Vercel React Best Practices 或 Next.js 性能优化技能，让 AI 助手按照行业最佳实践编写代码。

如果你正在使用 OpenClaw，直接前往 ClawHub.ai，通过 CLI 工具快速安装社区最热门的技能，扩展你的 Agent 能力。

如果你需要企业级集成，探索 AgenticSkills.io 的 MCP 服务器目录，将你的 AI Agent 连接到 GitHub、Stripe、AWS 等外部系统。

如果你专注营销且喜欢开源，查看 Marketing-Skills.com 和相关 GitHub 仓库，获取 160+ 免费营销技能，从 SEO 到增长工程一应俱全。

Agent Skills 正在重新定义我们与 AI 的协作方式——从”每次都要教”到”一次安装，持续复用”。选择合适的平台，让你的 AI Agent 从通用助手进化为领域专家。
