ChatGPT access denied 访问被拒绝，如何解决？

最近尝试登录，发现遇到了 ChatGPT access denied 访问被拒绝的提示，那怎么解决呢？是因为最近 openai 设置了相关验证条件：

You do not have access to chat.openai.com.
The site owner may have set restrictions that prevent you from accessing the site.

如何解决 ChatGPT access denied 访问被拒绝

浏览器层面，解决 access denied 错误

  • 设置英文。比如 chrome 浏览器设置 - 语言，把 English 置顶（火狐同理）
  • 关闭浏览器定位功能
  • 清除浏览器缓存和 cookie ：隐私和安全 > Cookie 和其他站点数据 > 查看所有站点数据和权限。你只需要找到浏览器右上角的三个点，然后在设置里找到清除缓存的选项就可以了。另外，你也可以在浏览器设置里选择清除浏览数据和缓存等选项来清除缓存
  • 禁止浏览器插件，最好是浏览器新增一个用户
  • 要么换个浏览器：比如火狐

网络层面，解决 access denied 错误

大概率都是网络层面的问题，和你梯子没有开全局代理就直接访问有关系

  • 解决一：设置全局代理即可
  • 解决二：由于代理的节点共享，太多人使用也是个问题。那么切换节点，比如新加坡、韩国、法国、英国。注意 HK 不行（有些机房 IP 也被拉入了黑名单）支持的国家如下

巴基斯坦、帕劳、巴勒斯坦、巴拿马、巴布亚新几内亚、秘鲁、菲律宾
波兰、葡萄牙、罗马尼亚、卢旺达、圣基茨和尼维斯、圣卢西亚、圣文森特和格林纳丁斯、萨摩亚等等

实在不行，换个浏览器或重置下 IP ipconfig /flushdns、netsh winsock reset和netsh int ip reset
