🔴 严重 必须立即修复

#1 Canonical 指向 GitHub Pages 域名
首页 canonical 标签指向 vincentkubo.github.io/colaos-website/zh/， 而非 colaos.ai/zh/。Google 会将 GitHub Pages 当成主站， colaos.ai 被降权为重复内容。
字段当前值（错误）应改为
canonical https://vincentkubo.github.io/colaos-website/zh/ https://colaos.ai/zh/
hreflang="en" https://vincentkubo.github.io/colaos-website/ https://colaos.ai/
hreflang="zh" https://vincentkubo.github.io/colaos-website/zh/ https://colaos.ai/zh/
hreflang="x-default" https://vincentkubo.github.io/colaos-website/ https://colaos.ai/
影响范围：首页 /zh/ 和英文首页 / 两页都受影响。
影响：上线至今的所有外链权重、品牌搜索，全部导给了 GitHub Pages，colaos.ai 自身几乎零权重积累。
#2 Sitemap 全站 URL 域名错误
Sitemap（https://vincentkubo.github.io/colaos-website/sitemap.xml） 中所有 <loc> 都使用 GitHub Pages 域名。Google 从 sitemap 抓到的 所有 URL 都会索引到错误的域名。 
<!-- 当前 sitemap 内容（截取） -->
<loc>https://vincentkubo.github.io/colaos-website/</loc>
<loc>https://vincentkubo.github.io/colaos-website/zh/</loc>
<loc>https://vincentkubo.github.io/colaos-website/about.html</loc>
<loc>https://vincentkubo.github.io/colaos-website/privacy.html</loc>
<loc>https://vincentkubo.github.io/colaos-website/terms.html</loc>
#3 robots.txt Sitemap 引用错误
robots.txt 中的 Sitemap: 指向 GitHub Pages 的 sitemap， 与 #2 形成连环错误。Google 从 robots.txt 发现 sitemap，sitemap 里又全是错误域名。 
# 当前 robots.txt
User-agent: *
Allow: /
Sitemap: https://vincentkubo.github.io/colaos-website/sitemap.xml  ← 错误！

# 应该改为
Sitemap: https://colaos.ai/sitemap.xml

🟡 中优 建议优化

#7 外部链接全部无 nofollow
证言区的所有即时 Jike 链接（web.okjike.com）和 X/Twitter 链接均未加 rel="nofollow" 或 rel="ugc"。 Google 会认为你在给这些社交帖子"投票"，稀释自身页面权重。
#8 所有 CTA 按钮指向同一个锚点
"下载 Mac 版"、"下载 Windows 版"、"免费开始"、"试用 ColaOS"、"免费获取 ColaOS"、 "立即下载" —— 共 8 个 CTA 按钮全部指向 #download 锚点。 Mac 和 Windows 下载没有区分链接，用户体验和埋点都无法区分。
#9 证言区内容大量重复
页面有 3 组轮播式证言区块（视觉上像是同一批用户的不同展示）， 共 36 个链接，实际内容大量重复。对 SEO 无增量价值，且显著增加页面体积（HTML 约 59KB）。 建议精简到一组高质量证言。
#10 缺少 robots meta 标签
虽然没有 robots meta 时默认允许索引，但建议显式声明： <meta name="robots" content="index, follow">
#11 子页面内容极薄
about（13.6KB）、privacy（1.9KB）、terms（1.9KB）内容很少。 并且 about/privacy/terms 页面似乎没有 hreflang 双语标注， sitemap 中也只列了它们的英文 .html 路径。
#12 docs 子域名未纳入 sitemap
文档站 docs.colaos.ai（nginx/1.27.5，独立部署）不在 sitemap 中， 也没有被 robots.txt 引用。建议为其单独配置 sitemap 或通过主站 sitemap index 引入。
